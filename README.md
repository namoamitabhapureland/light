## 專案介紹

- 一個純粹溫暖的**匿名心事分享空間**。  
- 在深夜裡，任何人可以放心傾訴、尋求鼓勵、獲得共鳴。  
- 以「點燈」表達支持，讓每一個孤單的人感受到「你不孤單」。
- 在這裡，任何人可以在深夜 anonymously 傾訴心事、尋求共鳴、獲得鼓勵，或只是隨意呢喃。  
- 完全匿名、沒有個資、沒有廣告，只有柔和的燈光與陌生人的善意陪伴。
- 網站以佛教「無量光」與「彌陀淨土」意象為靈感，將「點燈」作為支持他人的方式，讓每一個在黑暗中感到孤單的人，都能感受到「你不孤單」的溫暖。
- 發文後的內容會暫存於「燈火牆」，僅顯示最新 50 則，7天之後會自動物理消散，確保空間輕盈、不留負擔。

## 特色

- **完全匿名**：無需登入、無追蹤、無個資紀錄
- **四種發文類型**：
  - 心事樹洞（放心說，沒有人批判）
  - 求個鼓勵（你已經做得很好了）
  - 尋找共鳴（原來我不孤單）
  - 深夜呢喃（隨便說說，隨風消散）
- **燈火牆**：即時顯示最新 50 則動態，可為他人「點燈」表達支持
- **社群共治**：依賴群眾檢舉隱藏不當內容，無預先審核
- **短暫記憶**：內容 50 則後自動消散，減少長期負面累積
- **緊急資源**：提供台灣 1925 安心專線與 1995 生命線
- **純靜設計**：無廣告、無彈窗、深夜友好介面
- **免費開放**：永遠免費，純粹為需要溫暖的人存在

## 技術實現

- 純靜態 GitHub Pages 部署（HTML + CSS + JavaScript）
- 資料暫存於前端，無後端伺服器、不儲存個資
- 響應式設計，支援手機與桌面瀏覽
- 即時互動（點燈、回應、即時計數）

## 使用方式

1. 進入 [網站](https://namoamitabhapureland.github.io/light/index.html)
2. 選擇您今天需要的類型
3. 匿名輸入心事（0-1000 字）
4. 發出後即可在燈火牆看見，並接受他人的點燈與回應
